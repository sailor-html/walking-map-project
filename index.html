My Walking & Running Map Project

Exploring movement, place, and experience through mapping

This project explores my walking routes through spatial data and photography.

ArcGIS Web Map

This map displays my walking routes and collected photo locations using ArcGIS Online. It provides a spatial overview of where I have traveled, highlighting patterns in movement and areas where I tend to take photos. The goal is to visualize my everyday environment through geographic data.

Interactive Photo Map

This interactive map uses Mapbox GL JS to display my walking routes alongside geotagged photos. Users can click on points to view images taken during each walk, creating a more immersive experience that connects spatial data with personal observation.

Frequently Updated Dashboard

This dashboard summarizes my walking data through indicators and visual elements. It provides insights into the number of photos collected and spatial distribution, allowing users to quickly understand patterns and trends in my movement.
